
# Vim hacking reference

- Quick ref
  - `vim --help | less`
- General ref
  - vim man page: `man vim`
  - vim user manual TOC: `vim --cmd "help usr_toc.txt| only"`
- Specific, useful things
  - Shell in a window: `vim --cmd "help shell-window| only"`
  - Folding: `vim --cmd "help usr_28.txt| only"`
